Project: Foody Mini Project

This project involves the development of a mini web application related to the food domain. It is a web application created using HTML, CSS, and JavaScript. It encompasses features such as user authentication, an interactive menu browsing interface, dynamic order placement with real-time updates, and API integration for order tracking and status updates. The project provides a platform for users to browse menus, place orders, and track their orders seamlessly. I used technologies such as HTML, CSS, and JavaScript along various framework for the backend.
